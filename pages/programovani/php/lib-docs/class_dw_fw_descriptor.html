DwFwDescriptor::$bAutoRequested = false

If true, this view or action will be requested automatically.

DwFwDescriptor::$bDefault = false

If true, and if no other view was requested explicitly, this view or action will be requested automatically.

DwFwDescriptor::$bDontTrace = false

If true, this view won't be traced; means it won't be stored in the session and when some Action wants to go "back" (like "User Login Done" page), it will not return to this page, but some preceding.
