& cDBAccess::Select ( $  sSQL  )  [private]

Makes a SELECT and returns the result. Result returned by reference to let the object know when you unset($res); .

Návratová hodnota:
Depends on select mode set by SetSelectMode(): CDBA_SELECT_RETURNS_NULL_ON_ERROR: Returns null on error. CDBA_SELECT_RETURNS_CRESULT_ON_ERROR: Returns cDBA_Result object with error info.
