Dw Snippet Cache Written by Ondrej Zizka - http://ondra.zizka.cz , ondra@dynawest.cz, http://www.dynawest.cz

Caches portions of code - automates output buffer and include operations.

Example of use: ?><? $oDwCacheSnippet = new cDwCacheSnippet('page1_content.html'); if(!$oDwCacheSnippet->PrintFromCache()){ echo 'Hello everybody! This content was generated on '.date('j.n.Y H:i:s').'.'; $oDwCacheSnippet->Save(); } /* First, the cache object is created. Second, PrintFromCache() tries to print the cached data. If not found [or expired - may be implemented in future version], all output (echo'ed or the code outside PHP's "<? ? >" tags) is being caught until the method Save() is called. Save() stops catching, flushes the output to the client and saves the output trough the storage handler. Flushing can be suppressed by setting $bDirectEcho = false; In this example, data would be stored in 'page1_content.html' file.

Default behavior is to store data in files in the scripts directory, using pre-created cDwCacheStoreFiles class. That directory can be changed by providing another such object: $oDwCacheSnippet = new cDwCacheSnippet('page1_content.html', new cDwCacheStoreFiles('./pre/') ); The mean of data storage can be changed by extending the cDwCacheStore class.?><? Tries to print the cache content. If not exists [or has expired], starts caching. Returns false if the content was not found. Returns true if the content was found and printed. Returns the cache content if it was found but not printed ($bDirectEcho == true).
