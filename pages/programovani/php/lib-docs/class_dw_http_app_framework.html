Detailní popis

Dynawest HTTP Application Framework

A Framework for HTTP applications written in PHP 5.

Intended and designed to handle HTTP application flow in a similar way as J2EE technologies (esp. JSP) do it.

Paradigm

Each HTTP application has moreless this flow:

  • Load options
  • Initialize resources
  • Read inputs
  • Perform some actions, according to the input, options, and other conditions
  • Prepare some data that will be printed
  • Print the output, using the data prepared
  • Perform actions that should be performed after the data are printed
  • [Release resources, clean up, etc.]

This framework follows this logic.

What belongs to the framework core:

Everything that concerns whole application and affects all its parts:

  • Handling of the communication protocol, like HTTP headers (redirects etc.)
  • Application flow - actions, views, etc.
  • Options

What does not belong to the core and should be separate module (class):

Everything else:

  • User management
  • Database management
  • Object persistence
  • Internationalization
  • Logging
  • etc. etc...
